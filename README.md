# Customer Behavior Analysis

An end-to-end data analytics project that explores customer purchasing behavior using **Python, MySQL, and Power BI**. The project focuses on cleaning and analyzing retail transaction data to uncover meaningful business insights and present them through an interactive dashboard.

## Tools & Technologies

- Python (Pandas)
- MySQL
- Power BI
- VS Code
- GitHub

## Project Workflow

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- SQL Business Analysis
- Interactive Power BI Dashboard
- Business Insights & Recommendations (Report and presentation)

## Key Highlights

- Cleaned and transformed customer shopping data.
- Performed exploratory data analysis using Python.
- Solved business questions using SQL.
- Built an interactive Power BI dashboard.
- Generated actionable business insights to support data-driven decision-making.
